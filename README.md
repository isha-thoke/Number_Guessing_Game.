# 🎯 Number Guessing Game

A simple interactive game built with **HTML, CSS, and JavaScript**.  
Guess the secret number between 1 and 100 and see how many attempts you take!

---

## 🚀 Features
- Random number generation
- User input validation
- Feedback messages (Too high / Too low / Correct)
- Restart button for a new game
- Styled UI with gradient background

---

## 📂 Project Structure
- `index.html` → Game structure  
- `style.css` → Styling and layout  
- `script.js` → Game logic
